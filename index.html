下载

推荐项由浏览器自动识别。识别失败时，手动选择你的系统即可。

Windows
manual

适合 Windows 10/11。下载 ZIP 后解压，双击 Start-HermesGuiLauncher.cmd 打开中文图形启动器。

下载 Windows 安装包
数据目录：%LOCALAPPDATA%\hermes
包含：HermesGuiLauncher.ps1 / Start-HermesGuiLauncher.cmd
macOS
manual

适合 macOS。下载 .tar.gz 后解压，运行 HermesMacGuiLauncher.command。它是轻量控制面板：一级只保留安装、模型配置、本地对话、消息渠道、消息网关和维护入口，安装阶段直接走官方 install.sh。这种格式会保留执行权限，避免解压后提示“没有正确的访问权限”。首次运行时，macOS 仍可能要求你在系统安全设置中确认打开。

下载 macOS 安装包
数据目录：~/.hermes
包含：HermesMacGuiLauncher.command
建议：首次打开若被拦截，请在“系统设置 → 隐私与安全性”中允许执行

怎么用

不要把安装器当成 Hermes 的替代品。它只是图形入口，实际安装和运行仍然由 Hermes 官方命令完成。

1

下载

打开本页，使用推荐下载，或手动选择 Windows / macOS 包。

2

安装

打开图形启动器，点击安装或更新。命令会在独立终端中执行，方便观察进度。

3

检查

安装后运行 hermes doctor，再按需要执行配置向导、启动 Hermes 或卸载重装。

说明

页面和启动器尽量保持透明：能看到安装命令、能找到配置目录、能保留日志。

启动器提供什么

  • 安装或更新 Hermes Agent。
  • 运行配置向导、启动 Hermes CLI、执行 hermes doctor。
  • 打开 config.yaml、.env 和日志目录。
  • 执行标准卸载或彻底卸载，便于重装。
