安装后这样开始

Hermes 真正能用，关键不是“装完”，而是“配好模型并开始第一轮对话”。

1

完成安装

启动器会调用 Hermes 官方安装流程，并自动处理大部分依赖。通常只需要先确认 Git 已可用。

2

配置模型

先完成至少一个模型提供商配置。没有模型，Hermes 无法正常开始对话。

3

开始使用

建议先用本地对话确认可用；如果你希望通过微信、Telegram 等外部渠道继续收发消息，再继续配置消息渠道。

常见问题

这里只保留真正会影响你是否能开始使用的关键信息。

安装器会自动安装依赖吗？

会。根据 Hermes 官方安装文档，安装器会自动处理 Python、Node.js、ripgrep、ffmpeg 等常见依赖。通常只需要先确认 Git 已可用。

为什么安装后还要配置模型？

因为 Hermes 至少需要一个可用的模型提供商，才能真正开始对话。

消息渠道是必须的吗？

不是。普通用户可以先完成模型配置，直接开始本地对话。只有希望通过微信、Telegram 等外部入口持续收发消息时，才需要继续配置消息渠道。

我之前用过 OpenClaw，怎么办？

如果你本机已有旧的 OpenClaw 数据，可以在安装完成后使用迁移功能导入。迁移是可选步骤，不影响新用户直接开始使用。

重装会丢配置吗？

不一定。根据官方更新与卸载文档，Hermes 配置目录可以选择保留，方便后续重装恢复。

安装失败后该看哪里？

优先看启动器里的日志和反馈信息。如果需要核对官方步骤，再打开下方的官方资料链接逐项检查。
